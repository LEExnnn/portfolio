# 个人专属炫酷简历网站 - 任务确认单

## 1. 核心目标
在内卷的求职环境中脱颖而出，打造一个**高度定制化、视觉酷炫、充满动态高级感**的个人专属简历展示网站。项目将在 GitHub 托管，并通过 Vercel 此类平台实现自动部署上线。

## 2. 核心功能及模块

### 2.1 基础信息与视觉表现
- **整体风格:** 酷炫、高级、具备流畅的交互动态效果，摆脱传统简历的死板刻板印象。
- **个人简介区:** 提取历史 HTML 简历中的**核心内容**进行重新排版展示（抛弃旧 HTML 格式）。

### 2.2 核心展示模块
- **视频化项目介绍:** 支持嵌入或上传提前录制好的项目演示视频，支持点击播放。
- **项目与工具橱窗:** 展示个人项目、独立开发的工具集（重点展示 Vibe Coding 理念及 AI Agent 的开发成果）。

### 2.3 特色交互：个人专属 AI 问答系统 (Q&A Agent)
- 结合您的个人简历背景、工作经历及项目细节，打造一个基于网页的 AI 问答系统。
- 访客（如 HR / 面试官）可直接在网站上提问：“有方在上一家公司承担了什么核心工作？”，AI 将以您的视角的最佳解答自动回复。

## 3. DevOps 部署方案
- **代码管理:** GitHub 仓库。
- **自动化部署:** 采用 Vercel (即您提到的类似 "water" 的云托管平台，Vercel 完美支持静态及 Serverless 函数的免费托管部署)。

## 4. UI/UX 要求
- **语言:** 界面保持**全中文**（遵循全局 UI 铁律）。
- **动效标准:** 引入高级 CSS 动画或轻量级 3D/视差库（如 Framer Motion 或 Three.js 基础运用）以凸显“高级感”。
